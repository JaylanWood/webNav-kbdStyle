## 效果预览图

![预览图](https://github.com/JaylanWood/webNav-kbdStyle/raw/master/img/wallpaper.jpg)

## 简介

一个键盘风格的导航网站。

## 效果预览

[预览地址](https://jaylanwood.github.io/webNav-kbdStyle/)

## 特性

1. 搜索栏
2. 按下实体键盘，跳转对应的网站。
3. 导航网页的键盘会自动抓取目标网站的 icon 图标。
4. 鼠标移入键盘区，会浮现编辑按钮，可自定义目标网站域名（例如：baidu.com)。
